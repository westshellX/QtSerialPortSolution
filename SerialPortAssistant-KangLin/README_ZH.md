7. 编译
    - 用 QtCreator 编译
      * 直接用 QtCreator 打开工程文件 ![SerialPortAssistant.pro](SerialPortAssistant.pro) 
      * 直接编译就可以生成程序
    - 用命令行编译
      * 在项目根目录下建立 build 目录
      * qmake 生成编译工程
      * make install 用本地工具链生成程序
      * 在 install 目录下会生成程序

    ```sh
    mkdir build
    cd build
    qmake ../SerialPortAssistant.pro
    make install
    cd install #进入生成的程序目录
    ```
